## ğŸš€ Getting Started

### Prerequisites
- Python 3.6 or higher
- tkinter (usually included with Python)

### Installation

1. Clone this repository to your local machine:
```bash
git clone https://github.com/yourusername/aiwave.qt.git
cd aiwave.qt
```

2. No additional packages required - all programs use Python standard library!

### Running Programs

Navigate to any directory and run a Python file:

```bash
# Run a simple program
python3 Python_Beginner_Projects/fizzbuzz.py

# Run the calculator
python3 Python_Beginner_Projects/calculator.py

# Run the guessing game
python3 Python_Beginner_Projects/random_guessing_game/game.py

# Run GUI applications
python3 Python_Beginner_Projects/gui/text_editor/text_editor.py
```
